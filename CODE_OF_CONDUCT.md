# Code of Conduct

## Our Standard

We want SkillForge to be a project where people can learn, contribute, and disagree constructively without being treated badly.

Please bring:

- respect
- curiosity
- patience
- clear technical feedback

Please avoid:

- personal attacks
- harassment
- exclusionary or demeaning language
- performative hostility
- publishing private information without consent

## Scope

This applies to project discussions, issues, pull requests, review comments, and other public community spaces connected to SkillForge.

## Enforcement

Maintainers may edit, hide, lock, or remove content that makes the project less safe or less constructive, and may block repeat offenders when needed.

## Reporting

If someone's behavior is making the project feel unsafe, use the same contact path described in [SECURITY.md](SECURITY.md) when possible, or ask a maintainer for a quieter path to report it.
